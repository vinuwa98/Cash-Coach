# CashCoach 💰

CashCoach is an Android Application developed in Android Studio using Java and Google Cloud Firestore.

## Purpose
In today’s era, there are many small day-to-day expenses along with the usual expenses like rents and utility bills. Money these days is also a little tighter than it used to be and thus we can’t always remember how exactly we spend our money.

Therefore, we decided to find an easier way to solve this problem. As the name itself suggests, this project is an attempt to manage our daily expenses in a more efficient, simple, and manageable way for daily use.

This application allows people to eliminate any unnecessary expenses and figure out if they are able to sustain themselves or if they need to adjust their budget. Typically, people are able to optimize their spending by making more informed decisions. 

Instead of keeping a physical diary or log of expenses, this application enables the user to not just keep control of their expenses but also to generate visual reports. This application attempts to free the user from the burden of manual calculation and securely tracks expenditure in the cloud.

## Scope
This application has strong market potential as it is usable by anyone willing to manage their expenses, track their loans, and aim to save for future investments.

## Development Environment
* **IDE:** Android Studio
* **Programming Language:** Java
* **Backend & Auth:** Google Firebase Authentication
* **Database:** Cloud Firestore 

## Operational Environment
* **RAM:** 1GB or more
* **ROM:** 100MB or more
* **Processor:** 1GHz Dual Core or higher
* **Android Version:** Android 9.0 or higher
* **Internet Connectivity:** Required for cloud-based syncing

## Features

### Major Features
* **Transaction Tracking:** Insert, View, and Delete Income and Expense data.
* **Loan Management:** Track money borrowed and mark loans as "Paid" as it automatically updates in expenses.
* **Smart Dashboard:** Real-time calculation of Total Balance.
* **Reports:** Statistical breakdown of expenses and income (Today, Weekly, Monthly).
* **History Filtering:** Search and filter data by specific dates or transaction types.

### Security & Authentication
* User Registration (Sign Up)
* Secure Login / Logout
* Cloud data isolation

*Note: Ensure you have an active internet connection to make sure the app works properly.*
